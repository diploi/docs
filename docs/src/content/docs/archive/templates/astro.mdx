**Astro** is a web framework for building content-driven websites like blogs, marketing,
and e-commerce. Astro is best-known for pioneering a new frontend architecture to
reduce JavaScript overhead and complexity compared to other frameworks.
If you need a website that loads fast and has great SEO, then Astro is for you.

![Astro](../../../assets/astro-logo-light-gradient.png)

SSR support with the Node.js adapter is preconfigured.
