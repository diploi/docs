import { Code } from 'astro-expressive-code/components';
import {diploiElementList, extractElement} from '../../../../components/elementUtils'
export const AllElements = await diploiElementList()
export const nueDetails = extractElement(AllElements,'nue')
export const nuePackage = `${nueDetails.url}/${nueDetails.versions.pop()}`

Nue.js is a minimal framework for composing modular web applications with fast SSR. Pick this component when you want Nue's lean runtime while keeping deployment automation simple. Diploi runs the build and serves the output so you can focus on your UI modules.

## Add to your project

Paste this entry into the `components` list in `diploi.yaml` to add Nue.js to your project.

<Code code={`components:
  - name: ${nueDetails.name}
    identifier: ${nueDetails.identifier}
    package: ${nuePackage}`} lang="yaml" />

:::note
- identifier must match the folder name you use for this component in your repo.
- You can import ENV from add-ons into components using env.include.

Example:

```yaml
env:
  include:
    - mariadb.*
```
:::
